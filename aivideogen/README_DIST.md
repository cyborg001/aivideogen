# aiVideoGen v8.7.0 - Versión Estable Portable

¡Bienvenido a la versión más estable y potente del motor aiVideoGen!

## Novedades v8.7.0:
- **Etiquetas Fonéticas (`[PHO]`):** Control total sobre la pronunciación sin afectar a los subtítulos.
- **Limpieza Inteligente:** Los metadatos de YouTube ahora se limpian automáticamente de etiquetas técnicas.
- **Subtítulos Dinámicos (`[SUB]`):** Sincronización perfecta de overlays de texto sobre el video.
- **Audio Ducking Pro:** Calibración optimizada para transiciones suaves entre voz y música.
- **YouTube Metadata Pro:** Subida automática persistente tras corregir NameErrors críticos.

## Cómo empezar:
1. Ejecuta `Start_App.bat`.
2. Ve a `Crear Proyecto`.
3. Sube tus imágenes a `media/assets/`.
4. ¡Carga uno de los guiones de ejemplo en `guiones/ejemplos_v8.5/` para ver la magia!

## Documentación detallada:
Consulta la carpeta `docs/` para el manual completo y las guías de efectos.
