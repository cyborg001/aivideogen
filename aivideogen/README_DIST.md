# aiVideoGen v8.5.0 - Versión Estable Portable

¡Bienvenido a la versión más estable y potente del motor aiVideoGen!

## Novedades v8.5.0:
- **Flujo Maestro:** Las historias fluyen mejor gracias a la integración inteligente de conectores narrativos.
- **Encuadre Pro:** El sistema ahora detecta personas en las imágenes y ajusta el zoom (Zoom Out) o paneo vertical automáticamente para que nunca se corten rostros.
- **YouTube Metadata Pro:** Subida automática con descripciones enriquecidas (fuentes, hashtags, capítulos).
- **Core Assets Incluidos:** Esta distribución ya viene con toda la librería de SFX, Músicas y Overlays cinemáticos.

## Cómo empezar:
1. Ejecuta `Start_App.bat`.
2. Ve a `Crear Proyecto`.
3. Sube tus imágenes a `media/assets/`.
4. ¡Carga uno de los guiones de ejemplo en `guiones/ejemplos_v8.5/` para ver la magia!

## Documentación detallada:
Consulta la carpeta `docs/` para el manual completo y las guías de efectos.
