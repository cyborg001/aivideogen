# Solucionario: Examen de Física I - Forma C

## Parte I: Teoría
1. **Física**: Ciencia que estudia la materia, energía, tiempo y espacio, y sus interacciones.
2. **Redondeo (Prod/Div)**: El resultado debe tener el mismo número de cifras significativas que la cantidad con menor número de cifras significativas en la operación.
3. **Magnitudes SI**: Longitud (m), Masa (kg), Tiempo (s), Temperatura (K), Intensidad de corriente (A).
4. **Notación Científica**: $4.5 \times 10^{-7}$. El exponente negativo indica que el punto decimal se desplazó hacia la derecha (número menor que 1).

## Parte II: Práctica
1. **2.5 km a pulgadas**:
   - $2.5 \text{ km} \times 100,000 = 250,000 \text{ cm}$
   - $250,000 / 2.54 \text{ cm/in} = \mathbf{98,425.2 \text{ in}}$ (redondeado de $98,425.19...$)

2. **1.5 gal/s a cm³/hr**:
   - $1.5 \text{ gal/s} \times 3785.41 = 5,678.115 \text{ cm}^3/\text{s}$
   - $5,678.115 \times 3600 \text{ s} = \mathbf{20,441,214 \text{ cm}^3/\text{hr}}$

3. **Notación Científica**: $(6.0 \times 10^{-4}) \times (2.5 \times 10^{-5})$
   - $6.0 \times 2.5 = 15.0$
   - $10^{-4} \times 10^{-5} = 10^{-9}$
   - Normalización: $1.5 \times 10^{-8}$
   - Resultado: **$1.5 \times 10^{-8}$**

4. **Cifras Significativas**: $12.45 \times 2.0$
   - Multiplicación bruta: $24.9$
   - El término con menos c.s. es $2.0$ (2 cifras significativas).
   - Resultado redondeado: **25**
