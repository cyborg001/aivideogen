# Estrategia de Contenido: Gancho Primero (Hook-First)

Este flujo de trabajo garantiza que cada pieza de contenido generada (especialmente YouTube Shorts) maximice la retención desde el primer segundo.

## 1. Reglas del Gancho (Hook-Pro)
- **Impacto Inmediato**: El primer enunciado debe durar menos de 2 segundos.
- **Tipos de Ganchos**:
    - **Fórmula "Sabías que" (Curiosidad)**: "Sabías que... [DATO IMPACTANTE]". Debe ser una afirmación asertiva, no una pregunta de relleno.
    - **Fórmula "Noticia" (Urgencia)**: "¡Confirmado! [EVENTO]", "¡Última hora! [EVENTO]". Tono de autoridad y primicia.
- **Dato Específico**: Prohibido usar introducciones genéricas. Empezar con un nombre propio, un número impactante o la marca directa.

## 2. Estructura del Script (Retención Extrema)
1. **Gancho (0-2s)**: Revelación directa con la marca o dato duro.
2. **Contexto Rápido (2-10s)**: Saludo de marca "Bienvenidos a NotIACi" + Por qué esto importa.
3. **Cuerpo (10-50s)**: Desarrollo con cambios visuales cada **2-4 segundos**.
4. **Cierre/CTA (50-60s)**: Pregunta específica para debate + suscripción.

## 3. Ejemplo de Transformación de Marca
- **Antes (Débil)**: "¿Sabías que la IA puede leer la mente?"
- **Después (Fuerte)**: "Sabías que la IA ya puede traducir tus pensamientos en texto con un 90% de precisión."

## 4. Aplicación Técnica
Cuando se use la herramienta de generación de scripts, siempre se debe validar que el primer párrafo cumpla con estas métricas de "Gancho de Alto Impacto".
